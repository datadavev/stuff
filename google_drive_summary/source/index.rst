Google Drive Metadata
=====================

This document contains a list of content available in a Google Drive folder. A list of folders, permissions associated with each folder, and the contents of each folder are provided.

Note that the permissions on a specific file may differ from the permissions
assigned to the file's folder.

Note that the depth of folders presented is limited to the top few levels.
Deeper passes can be generated on request.

The list is generated by a script and must be manually run.
